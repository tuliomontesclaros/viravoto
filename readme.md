# @viravoto

## Dependencies

You will need docker installed

## Running

Just run

```shell
docker-compose up
```

The service will run at localhost:3000

The site will run at localhost:80

## Structure

All html, css, javascript is on the `site` directory.

Most backend code is on `routes/records`.
